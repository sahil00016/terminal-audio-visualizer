## Data Shapes

### `cfg` dict (config.py)

```
{
  "theme":   str,    # one of THEMES
  "volume":  int,    # 0вҖ“100
  "shuffle": bool,
  "repeat":  bool
}
```

### MPV IPC message

```
Request:  {"command": list[str|float], "request_id": int}
Response: {"data": any, "error": "success"|str, "request_id": int}
Event:    {"event": str, ...}   (unsolicited, ignored)
```

### `bands` list

```
list[float]  length = width of visualizer area in terminal columns
             each value вҲҲ [0.0, 1.0]  (normalised amplitude)
```

### `.desktop` entry (Linux)

```
[Desktop Entry]
Name=Music TUI
Exec=music-tui %f
Terminal=true
Type=Application
MimeType=audio/mpeg;audio/flac;...
```

### Windows registry (Open With)

```
HKCU\Software\Classes\MusicTUI\shell\open\command вҶ’ "music-tui" "%1"
HKCU\Software\Classes\.mp3\OpenWithProgids\MusicTUI вҶ’ ""
... (repeated for each AUDIO_EXT)
```
